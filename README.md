# Quantifying Modality Contributions via Disentangling Multimodal Representations using Partial Information Decomposition (PID).

## Table of Contents

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/namitharayasam/vlm-pid-analysis.git
cd vlm-pid-analysis
```

2. (Optional) Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # Mac/Linux
# .\venv\Scripts\activate  # Windows
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```
